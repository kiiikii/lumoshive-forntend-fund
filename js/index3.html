Tentang Saya

Mengenal lebih dekat tentang keahlian dan passion saya

Halo! Saya seorang Front-End Developer dengan pengalaman 3 tahun dalam membangun website dan aplikasi web modern. Saya memiliki passion dalam menciptakan antarmuka yang intuitif dan responsive.
